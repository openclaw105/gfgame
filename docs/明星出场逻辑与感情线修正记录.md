# 明星出场逻辑与感情线修正记录

更新时间：2026-05-24

## 本次修正目标

- 每位明星出场必须有工作、活动或熟人引荐理由，避免突兀空降。
- 女主葵保持“小画家 / 临时画师 / 展区协助”身份，不再承担明星级热搜中心。
- 薇薇和龚俊线改为成年人直接沟通，女主只做朋友支持，不替薇薇表态。
- 肖战、宋威龙保留可选甜蜜分支，但必须建立在工作接触、边界清楚和玩家主动选择上。

## 关键改动

1. 第3章品牌布场
   - 龚俊首次正式出场改为品牌方对接人，和薇薇旧识的关系在工作现场自然交代。
   - 原“帮忙照看薇薇”改为“怕邀约让薇薇为难”，选择项改成让龚俊直接问薇薇意愿、先把工作收完、由薇薇自己决定。
   - 欧阳娜娜负责采访区调整，田曦薇负责短片空镜，肖战负责彩排动线，所有明星都有明确现场任务。

2. 第4章第一次会所
   - 宋威龙由卢昱晓引荐，递名片走正式片场速写委托，不是私人突兀邀约。
   - 肖战敬酒分支加入“换温茶、不用陪应酬”的甜蜜但克制处理。
   - 肖战工作室车接送分支改成侧门避镜头和到家提醒，更符合公众人物社交边界。

3. 第5章片场
   - 宋威龙线强化为正式工作委托后的片场再见。
   - “只画宋威龙”分支加入暖手宝和收工后可拒绝的吃饭邀请，张婧仪在场调侃，降低突兀暧昧感。
   - 片场最后一天新增分支：速写本复印件交接后，宋威龙正式表达杀青后再约，仍保留工作边界。

4. 第8章第二次会所
   - 原“带不带薇薇”整体改为“薇薇自己决定要不要去会所”。
   - 原“让龚俊接她”“给龚俊发对她好点”等代理式台词全部改为薇薇自己回消息、自己确认边界。
   - 龚俊敬薇薇时，玩家选项改为陪她举杯、帮她换茶、退开让他们自己聊，避免女主替薇薇处理恋爱关系。

5. 第10章终章
   - 龚俊感谢内容改为尊重薇薇选择，不再让女主用“你对她好点”监管关系。
   - 宋威龙机缘卡分支改为“杀青后如果愿意，不聊工作吃饭；不愿意也没关系”，让甜蜜线可选且不压迫。
